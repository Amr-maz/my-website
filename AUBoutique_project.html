AUBoutique: The AUB Marketplace

This marketplace app was created to showcase my skills in full-stack development. Built with Python and using MYSQL??, it constitutes a shared space for the AUB community (or any university or even other communities) to list items for sale and browse items listed by other AUB members in order to create a platform for members to find whar they're looking for at discounted prices and reduce the need to buy new products thus reducing unecesary waste. The app features authentication upon sign in, a user-friendly GUI, and the option to directly message the buyer/seller of the product of interest.

  • Mobile-first responsive layout
  • Interactive navigation menu
  • Gradient-themed dark mode design
  • Custom project showcase pages
